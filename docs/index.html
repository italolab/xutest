O que é?


O xutest é um framework para testes de unidade em C/C++ que pode ser utilizado em conjunto com o foxmake ou outro software de automação de build.

O xutest é uma biblioteca estática (static library) que pode ser incluída no executável do seu projeto, independentemente de você utilizar um software de automação de build.
