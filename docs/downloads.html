Download


Para fazer o download do xutest você pode escolher uma das seguintes opções:

Windows x64

Arquivo ZIP: xutest-1.0-windows-x64.zip

Linux x64

Arquivo DEB: xutest-1.0-linux-x64.deb

Arquivo TAR.GZ: xutest-1.0-linux-x64.tar.gz

O código fonte

Repositório : https://github.com/italolab/xutest

Arquivo ZIP: xutest-v1.0.zip

Arquivo TAR.GZ: xutest-v1.0.tar.gz

Como instalar?

Instalando no windows

Para instalar no windows, você pode extrair a versão zipada: xutest-x.x-windows-x64.zip (onde x.x é a versão) e copiar as pastas include e libs para a pasta raiz do mingw.

Instalando no linux

Se estiver no Ubuntu, ou outra distro derivada do Debian, pode fazer o download do pacote .deb: xutest-x.x-linux-x64.deb (Onde x.x é a versão) e executar o seguinte comando:

sudo dpkg -i xutest-x.x-linux-x64.deb

Se preferir a versão .tar.gz, basta baixar o arquivo: xutest-x.x-linux-x64.tar.gz, descompactar, e copiar o arquivo "libxutest.a" para pasta /usr/lib/x86_64-linux-gnu (isso caso esteja num derivado do debian), e copiar o conteúdo da pasta "include" para pasta /usr/include.

Não instalar

Você pode optar por não instalar o xutest e incluí-lo diretamente no seu projeto. Veja como fazer essa configuração em: incluir sem instalar.
